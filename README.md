# 聲易記 - 智慧語音記帳 App (Voice Expense Tracker)

這是一個基於 HTML5、CSS3 與原生 JavaScript 打造的**深色毛玻璃風格 (Glassmorphism)** 語音記帳 Web 應用程式。使用者只需按下麥克風說出日常消費內容，系統即可透過智慧自然語言解析（NLP）自動判定金額、消费品項與類別，並將數據轉化為精美的視覺化圖表。

---

## 🌟 核心特色

1. **🎤 免打字語音輸入**：
   - 使用 Web Speech API（`SpeechRecognition`）進行高精確度的繁體中文語音辨識。
   - 語音按鈕配有等離子光環與即時聲波動態反饋。
2. **🧠 智慧 NLP 解析引擎**：
   - 支援口語化中文國字數字（例如「兩百五」、「一萬二」、「三千」）以及標準阿拉伯數字與小數點的混用解析。
   - 具備關鍵字字典，自動將消費歸類至：**食 (飲食)、衣 (服飾)、住 (居家)、行 (交通)、育 (教育)、樂 (娛樂)、其他**。
3. **📊 精美 SVG 互動圖表**：
   - **類別圓餅圖**：具備 Donut 甜甜圈樣式與懸停微互動，鼠標移入時，中心會同步顯示該類別的消費總額與佔比。
   - **消費趨勢折線圖**：動態繪製最近 7 天的每日消費趨勢，並帶有漸層背景與亮點動畫。
4. **💰 月預算管理與警示**：
   - 可隨時修改每月預算上限。
   - 當預算使用率超過 80% 時進度條將顯示黃色警示，超過 100% 時會以亮紅色警告，提醒您克制消費。
5. **🔒 隱私安全（無伺服器）**：
   - 所有記帳紀錄完全透過瀏覽器的 `LocalStorage` 儲存在本地，絕不將個人財務隱私上傳至伺服器。
6. **⌨️ 智慧手動備份輸入**：
   - 若瀏覽器不支援語音（如 Firefox 舊版本）或麥克風權限被拒，可直接在手動文字欄輸入語句（如「晚餐吃火鍋花了550元」），同樣能觸發智慧解析引擎。

## 📁 檔案結構

專案完全由純前端靜態檔案構成，結構極其簡潔：
```text
voice-expense-tracker/
├── index.html   # 網頁結構、預算卡片、語音控制區與 Modal 彈窗
├── style.css    # 玻璃擬態 UI、漸層定義、聲波動畫與響應式切換
└── app.js       # 語音核心、LocalStorage 存取、NLP 解析與 SVG 繪圖邏輯
```

## 🚀 快速開始

### 1. 本地開啟
1. 下載專案並在檔案總管中雙擊 [index.html](file:///C:/Users/USER/.gemini/antigravity/scratch/voice-expense-tracker/index.html)。
2. 本地開發時，建議使用 IDE 的 `Live Server`（或終端機執行 `npx http-server`）以網域 `http://localhost:...` 方式載入，這樣在申請瀏覽器麥克風權限時最為穩定。

### 2. 語音解析範例
您可以點選介面中的提示字，或對麥克風說出以下內容進行測試：
- *「中午吃牛肉麵花了一百八十元」* ➔ 自動解析為：**「吃牛肉麵」**，金額 **`180`**，類別 **`食`**。
- *「買了一雙鞋子兩千五」* ➔ 自動解析為：**「鞋子」**，金額 **`2500`**，類別 **`衣`**。
- *「房租一萬二千元」* ➔ 自動解析為：**「房租」**，金額 **`12000`**，類別 **`住`**。
- *「搭計程車花了三百塊」* ➔ 自動解析為：**「搭計程車」**，金額 **`300`**，類別 **`行`**。

## 🛠️ 技術限制與相容性

- **語音辨識相容性**：Web Speech API 主要是由 Chrome、Edge 等 Chromium 核心瀏覽器完全支援。在語音不可用的瀏覽器中，按鈕會自動呈現停用狀態，此時您可以完全使用「手動輸入智慧解析」功能，不影響主要體驗。
- **網路需求**：部分瀏覽器的語音辨識依靠雲端語音服務，建議在連線狀態下使用以獲取最佳辨識速度與精準度。
